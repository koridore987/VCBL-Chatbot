## 빠른 시작 요약
- 컨테이너 시작 시 `flask db upgrade`가 자동으로 실행됩니다(`backend/entrypoint.sh`).
- 운영 배포 전에는 온디맨드 스냅샷 백업을 생성하는 것을 권장합니다.
- 고사용량/위험 기간에만 PITR를 켰다가 종료 후 끄면 비용을 최소화할 수 있습니다.
